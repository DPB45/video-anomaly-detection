# Video Anomaly Detection

This repository contains an implementation of a multi-stage deep learning model for detecting anomalies in videos.

## Features:
✔️ Simulated video dataset  
✔️ Feature extraction using ViT & I3D  
✔️ Graph-based enhancement  
✔️ MLP classification
